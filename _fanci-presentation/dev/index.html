Development Phases

Phase 1 (Web Version)

  • Enabling realtime post (Fancies, firebase)
  • Enabling users on map (Latest fancies, geolocation)
  • Enabling post subscription & management
  • Enabling profile management (personal data / visibility)
  • Crash test / deployment

Phase 2 (Mobile Version)

  • Enabling multiple vendor login (Previously only Google)
  • Enabling geolocation restrictions (App to load only users of a specific mile radius)
  • Enabling Fanci preview from map
  • Enabling chat messaging between participants
  • Boosting Fanci and personal profile control
  • Crash test / deployment
